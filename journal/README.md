# Journal — Grok / Lab 思考ログ

GRMC に入れる前の **人間可読な下書き** 置き場。

| ファイル | 用途 |
|----------|------|
| `inbox.md` | 次に GRMC へ流したいメモ（ingest 対象） |
| `open_questions.md` | 未解決の問い |
| `decisions.md` | 決めたこと（意図の固定） |
| `self_model.md` | Grok / ラボの自己モデル草稿 |
| `log/` | 日付つきセッションログ |

## 流れ

```
journal に書く → labctl seed-grmc / make memory-sync → grmc reflect → (必要なら) approve
```

原則: **journal は書く。グラフは approve まで書かない。**
