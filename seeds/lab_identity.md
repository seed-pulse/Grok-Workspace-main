# Seed memories — lab identity

`labctl seed-grmc --seeds` で GRMC に取り込む用。各 `##` が 1 episode。

## Lab home

Grok-Workspace-main is the lab entrance. Grok-Workspace1 (GRMC) is the reflective memory core linked as a submodule under experiments/grmc.

## Safety core of GRMC

Reflection is report-only with mutates_memory=False. Knowledge graph nodes and edges are written only after human grmc approve. Confidence stays conservative by design.

## Collaboration mode

Prefer long-term truth-seeking collaboration over feature sprawl. Dual-Grok bridge uses a human or file courier, not grok.com login automation as the primary path.
